Start from clean install of Debian 13

Creates a Caddy instance, a portal to unify everything and an openwebui instance to chat with the data

##### Need a clean install of Apache Superset

```bash
git clone https://github.com/CheezeLover/Hyperset.git
```
```bash
cd Hyperset
chmod u+x setup_podman.sh
```
Update .env file with the right domain
```bash
./setup_podman.sh
```
