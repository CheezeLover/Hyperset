Hyperset Documentation

This is a static documentation page — no backend needed.

Drop an index.html file in Pages/docs/ and it will appear as a panel in the portal.

Adding a new page

  1. Create a subfolder under Pages/, e.g. Pages/my-tool/
  2. Add an index.html — this is the page shown in the iframe.
  3. Optionally add a backend.py exposing a FastAPI router = APIRouter().
  4. The page appears automatically in the portal sidebar within seconds.

Calling the backend from your page

Your index.html can call its backend at:

https://pages.{domain}/{page-name}/api/...
